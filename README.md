# Leaf & Root – E-Commerce Website

Leaf & Root is a full-stack e-commerce web application for selling plant-based products such as plants, seeds, and gardening items.
This project was developed as a Semester 5 academic project.

---

##  Features

* User registration and login
* Browse plant products
* Add products to cart
* Place orders
* Product management APIs
* Responsive frontend interface

---

##  Tech Stack

**Frontend**

* React
* Vite
* CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB Atlas

##  Installation

### 1. Clone the repository

git clone https://github.com/sevika20/leaf-and-root.git

### 2. Install dependencies

Frontend

cd leaf
npm install

Backend

cd ../root
npm install

### 3. Add environment variables

Create a `.env` file in the backend folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

### 4. Run the project

Run backend

node index.js

Run frontend

npm run dev
